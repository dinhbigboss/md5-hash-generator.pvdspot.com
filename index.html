Frequently Asked Questions

What is MD5?
MD5 (Message-Digest Algorithm 5) is a widely used cryptographic hash function that produces a 128-bit (16-byte) hash value, typically expressed as a 32-digit hexadecimal number.
Is it reversible?
No. MD5 is a one-way hash function. You cannot decrypt an MD5 hash back to the original text. However, "Rainbow Tables" can be used to lookup common hashes.
Is my data sent to a server?
Absolutely not. This tool uses JavaScript to generate the hash locally in your browser. Nothing leaves your device.
